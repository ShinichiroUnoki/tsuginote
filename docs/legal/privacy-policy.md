# TsugiNote プライバシーポリシー

**最終更新日**: 2026年3月29日
**施行日**: 2026年3月29日

---

## 1. はじめに

TsugiNote（ツギノート）（以下「当社」）は、ユーザーの個人情報の保護を重要な責務と認識し、個人情報の保護に関する法律（以下「個人情報保護法」／APPI）および関連法令を遵守します。

本プライバシーポリシー（以下「本ポリシー」）は、当社が提供するTsugiNoteサービス（以下「本サービス」）における個人情報の取り扱いについて定めます。

---

## 2. 収集する個人情報

### 2-1. アカウント登録時に収集する情報

| 情報 | 目的 | 必須/任意 |
|------|------|----------|
| メールアドレス | アカウント認証、通知送信 | 必須 |
| パスワード（bcryptハッシュ化して保存） | アカウント認証 | 必須 |
| 氏名 | サービス内表示 | 必須 |
| 会社名 | ワークスペース管理 | 任意 |
| アバター画像 | サービス内表示 | 任意 |

### 2-2. サービス利用時に自動収集する情報

| 情報 | 目的 |
|------|------|
| IPアドレス | セキュリティ監視、不正アクセス検知 |
| ブラウザ情報（User-Agent） | サービス最適化 |
| アクセスログ（日時、操作内容） | サービス改善、障害対応 |
| Cookie / セッション情報 | セッション管理、認証状態維持 |

### 2-3. 決済情報

- クレジットカード情報は、当社のサーバーには保存されません。
- 決済処理は全てStripe, Inc.に委託しており、Stripe社のプライバシーポリシーおよびPCI DSS準拠のセキュリティ基準に従います。
- 当社が保持するのは、Stripeが発行する顧客ID・サブスクリプションIDのみです。

### 2-4. ユーザーが入力するコンテンツ

- ナレッジドキュメント、AI生成依頼テキスト、検索クエリ等
- これらのコンテンツはサービス提供のために保存・処理されますが、個人情報保護法上の「個人情報」に該当する場合は、本ポリシーに従い適切に取り扱います。

---

## 3. 個人情報の利用目的

当社は、収集した個人情報を以下の目的で利用します。

1. 本サービスの提供・運営
2. ユーザー認証・アカウント管理
3. 料金の請求・決済処理
4. サービスに関する通知（障害情報、メンテナンス、重要な変更）
5. ユーザーからのお問い合わせ対応
6. サービスの改善・新機能開発のための統計分析（匿名化処理後）
7. 不正利用の検知・防止
8. 法令に基づく対応

上記以外の目的で個人情報を利用する場合は、事前にユーザーの同意を取得します。

---

## 4. AI処理におけるデータの取り扱い

### 4-1. AI生成機能

1. ユーザーが入力したテキストは、AI処理のためにLLM APIプロバイダー（OpenAI、Anthropic等）に送信されます。
2. 送信されるデータは、AI生成に必要な最小限のテキストデータに限定されます。
3. LLM APIプロバイダーのデータ取り扱いは、各プロバイダーのプライバシーポリシーに従います。

### 4-2. セマンティック検索（RAG）

1. ドキュメントのベクトル埋め込み（embedding）は、当社管理のベクトルデータベースに保存されます。
2. ベクトルデータはワークスペースごとに分離され、他のワークスペースからアクセスすることはできません。

### 4-3. AI学習への不使用

- **当社は、ユーザーのコンテンツをAIモデルの学習・トレーニングに使用しません。**
- LLM APIプロバイダーに対しても、API利用データをモデル学習に使用しないプラン（例: OpenAI API、Anthropic API）を利用します。

---

## 5. 個人情報の第三者提供

当社は、以下の場合を除き、ユーザーの個人情報を第三者に提供しません。

1. **ユーザーの同意がある場合**
2. **法令に基づく場合**（裁判所の命令、捜査機関からの照会等）
3. **サービス提供に必要な業務委託先**（以下に限定）:

| 委託先 | 委託内容 | データの種類 |
|--------|---------|-------------|
| Stripe, Inc. | 決済処理 | 顧客ID、決済関連情報 |
| LLM APIプロバイダー（OpenAI / Anthropic） | AI生成処理 | ユーザー入力テキスト（一時的） |
| クラウドインフラプロバイダー | サーバーホスティング | 暗号化された全データ |
| メール配信サービス | トランザクションメール | メールアドレス、氏名 |

業務委託先とは、個人情報の適切な取り扱いに関する契約を締結し、適切な監督を行います。

## 6. データの保管・保持

### 6-1. 保管場所

| データ種別 | 保管場所 |
|-----------|---------|
| アプリケーションデータ | 日本国内またはアジアリージョンのクラウドサーバー |
| ベクトルデータ | 同上 |
| バックアップデータ | 同上（暗号化保存） |
| 決済データ | Stripe社のセキュアサーバー（PCI DSS準拠） |

### 6-2. 保持期間

| データ種別 | 保持期間 | 根拠 |
|-----------|---------|------|
| アカウント情報 | アカウント存続中 + 解約後30日 | サービス提供に必要 |
| ナレッジコンテンツ | アカウント存続中 + 解約後30日 | データエクスポート期間 |
| アクセスログ | 90日間 | セキュリティ監視・障害対応 |
| 決済履歴 | 7年間 | 税法に基づく保存義務 |
| お問い合わせ記録 | 3年間 | カスタマーサポート品質維持 |

### 6-3. データ削除

- 解約後30日経過後、ユーザーのコンテンツデータは完全に削除されます（物理削除）。
- バックアップからの削除は、バックアップの保持期間（7日間）経過後に自動的に行われます。
- ユーザーは、解約前にデータエクスポート機能を使用してデータを取得できます。

---

## 7. データのセキュリティ

当社は、個人情報の漏洩、滅失、毀損を防止するため、以下のセキュリティ対策を実施します。

### 7-1. 技術的措置

| 対策 | 詳細 |
|------|------|
| 通信暗号化 | TLS 1.3による全通信の暗号化 |
| 保存時暗号化 | AES-256によるデータベース暗号化 |
| パスワード保護 | bcryptハッシュ化（ソルト付き） |
| アクセス制御 | ワークスペース間のデータ完全分離 |
| 認証 | JWT + リフレッシュトークン |
| レート制限 | APIエンドポイントごとのレート制限 |
| 脆弱性管理 | 依存パッケージの自動更新（Dependabot） |

### 7-2. 組織的措置

- 個人情報へのアクセスは、サービス運営に必要な最小限に限定
- アクセスログの定期的な監査
- セキュリティインシデント対応手順の整備

## 8. ユーザーの権利

個人情報保護法に基づき、ユーザーは以下の権利を有します。

| 権利 | 内容 | 対応方法 |
|------|------|---------|
| 開示請求 | 保有する個人情報の開示 | 設定画面またはメールで請求 |
| 訂正請求 | 誤った個人情報の訂正 | 設定画面で直接修正 |
| 削除請求 | 個人情報の削除 | アカウント削除で対応 |
| 利用停止請求 | 個人情報の利用停止 | メールで請求 |
| データポータビリティ | データのエクスポート | 設定画面からJSON/CSV形式でエクスポート |

開示・利用停止請求への対応期間: 請求受領後14日以内

---

## 9. Cookieの使用

### 9-1. 使用するCookie

| Cookie種別 | 目的 | 保持期間 |
|-----------|------|---------|
| セッションCookie | 認証状態の維持 | ブラウザ終了まで |
| 認証トークン | ログイン状態の維持 | 7日間（リフレッシュトークン: 30日間） |

### 9-2. 使用しないCookie

- 広告トラッキングCookieは使用しません。
- サードパーティのアナリティクスCookieは使用しません（将来導入する場合は事前通知します）。

---

## 10. 未成年者の個人情報

本サービスは、16歳未満の方を対象としていません。16歳未満の方が個人情報を提供したことが判明した場合、速やかに削除します。

---

## 11. 越境データ移転

AI生成処理のため、ユーザーが入力したテキストデータが日本国外のLLM APIプロバイダーのサーバーに一時的に送信される場合があります。

- 送信先: 米国（OpenAI、Anthropic等のサーバー所在地）
- データは処理後、APIプロバイダー側で保持されません（API利用規約に基づく）
- 個人情報保護法第28条に基づく適切な措置を講じます

---

## 12. プライバシーポリシーの変更

1. 当社は、法令の改正、サービス内容の変更等に応じて、本ポリシーを変更することがあります。
2. 重要な変更を行う場合は、変更の30日前までにメールまたはサービス内通知でお知らせします。
3. 変更後も本サービスを継続利用した場合、変更に同意したものとみなします。
